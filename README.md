# Cybersecurity-Labs Windows Security Controls & Audit Readiness
This repository contains hands-on cybersecurity labs focused on implementing, Validating, and documenting windows security control. The labs demonstrate both thechnical execution and governance risk and compliance (GRC) alignment through control testing, audit evidence generation and security validation. 

These excersises were completed in controlled lab enviroments and are designed to refelect real-world security, audit and operational scenarios.

-------

## Focus Areas
-Windows security hardening
-Preventive and detective security controls
-Encryption and data protection
-Authentication and account security
-Security loggin and audit readiness
-Evidence Preservation and compliance alignment

-------

## Frameworks & Standards Referenced
- Nist SP 800-53
- ISO/ IEC 27001
- CIS Critical Security Controls

-------

## Labs Included
1. Disk Encrytion & Data Protection (Bitlocker - Windows)
2. Local Security policy & Account Hardending
3. Account Lockout & Audit Loggin Configuration
4. Even Log Review & Evidence Preservation

-------

## Skills Demonstarted
- Security control implemenatation and validation
- Comman-line verification (windows)
- Policy interpretation and enforcement testing
- Audit evidence documentation
- Risk identification and remediation
- Preventive vs detective control analysis

-------

## Disclaimer
All labs were performed in isolation virtual enviroments. No real user data, Credentials, or production were used. Sensitive information has been sanitiezed or replaced with placeholders.
